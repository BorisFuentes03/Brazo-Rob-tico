¿Que es un Brazo Robótico?

Un robótico brazo es un sistema mecatrónico programablecompuesto por articulaciones y eslabones, diseñado para realizar tareas de manipulación, movimientos precisos o interacción con el entorno, generalmente mediante control automático (como lazo cerrado) y sensores de retroalimentación
